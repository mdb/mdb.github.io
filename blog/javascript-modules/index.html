JavaScript Modules

July 7, 2014

The next version of JavaScript features a module system. jsmodules.io offers a clear overview.

My notes…

Defining a module

// a file called hello.js
var hello;

hello = function () {
  console.log('hello');
};

export default hello;

Importing and using a module:

// a file called using.js
import hello from "hello";

hello();
// => hello

Named exports

A module with multiple exports:

// a file called hello.js
var hello,
    hola;

hello = function () {
  console.log('hello');
};

hola = function () {
  console.log('hola');
};

export default hello;
export var hola;

Grouped named exports:

export { hello, hola, bonjour };

function hello() {
  console.log('hello');
}

function hola() {
  console.log('hola');
}

function bonjour() {
  console.log('bonjour');
}

Named imports

Basic usage:

import { hola } from "hello";

hola();
// => hola

Renaming named imports:

import { hola as hi } from "hello";

hi();
// => hola

Importing all a module’s exports into a local namespace:

import * as greeter from "hello";

greeter.hello();
// => hello

greeter.hola();
// => hola
