Install `goss`:

```
curl -L https://github.com/goss-org/goss/releases/latest/download/goss-alpha-darwin-amd64 -o ~/bin/goss
chmod +x ~/bin/goss
```

Install `dgoss`:

```
curl -L https://github.com/goss-org/goss/releases/latest/download/dgoss -o ~/bin/dgoss
chmod +x ~/bin/dgoss
```
